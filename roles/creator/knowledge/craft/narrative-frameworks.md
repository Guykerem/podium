# Narrative Frameworks

Structures for scripts, articles, and threads. Pick one; don't invent a new structure per piece.

## PAS — Problem / Agitate / Solution

**When:** Short-form content (TikTok, Reels, Shorts, LinkedIn video). The fastest-converting framework.

**Structure:**
- **Problem** (hook, 0-3s): Name the pain specifically
- **Agitate** (3-15s): Make them feel it — specifics, stakes, personal angle
- **Solution** (15s+): Deliver the fix, the insight, the tool

**Example beats:**
```
P: "You're using three apps to ship one post."
A: "By the time you cut the clip, write the caption, design the cover, and schedule — the hook you had is cold. The trend has passed. You ship the thing, exhausted, at 8pm."
S: "This agent does the last mile for you. Capture once. Everything else is drafted by the time you sit down."
```

## AIDA — Attention / Interest / Desire / Action

**When:** Longer-form content with a specific conversion goal (newsletter, product intro, sponsored segment).

**Structure:**
- **Attention**: Hook
- **Interest**: Expand the problem, show context, credibility
- **Desire**: Paint the picture of the solved state
- **Action**: Clear CTA

Less common in pure editorial — more common in sponsored or lead-gen content.

## StoryBrand (Donald Miller)

**When:** You need the audience to feel like the hero, not admire you. Works for coaching, courses, consulting content.

**Structure:**
1. **A character** — the viewer is the hero
2. **has a problem** — name it specifically
3. **meets a guide** — you
4. **who gives them a plan** — concrete steps
5. **and calls them to action** — next step
6. **that ends in success / avoids failure** — paint both outcomes

Useful for long-form YouTube sold-by-the-creator content. The creator is never the hero in StoryBrand.

## Hero's Journey (Campbell / Vogler simplified)

**When:** Documentary-length content, personal narrative, episode-of-a-life videos.

**Simplified arc:**
- Ordinary world → Call to adventure → Refusal → Mentor appears → Crossing threshold → Trials → Transformation → Return with elixir

For short-form, compress to:
- Where I was → What happened → What I learned → What I do now

## MrBeast Retention Framework

**When:** Long-form YouTube, 8-20 min content.

**Rules:**
- **Hook in first 0:15** — promise + visual proof of stakes
- **New element every 40-60s** — visual change, new information, twist, cut to b-roll
- **Open loops** — tease something in minute 2 that you deliver in minute 8
- **Ramping stakes** — each segment > previous (more surprising, more visual, more at stake)
- **Retention graph informs re-edit** — drop cliffs get cut in the next version

## Atomic Essay / Ship 30 for 30 structure

**When:** Short-form text — LinkedIn, short newsletter, Substack Notes.

**Structure:**
- **Title / hook** — standalone claim
- **Opening line** — expand the claim with sharpness
- **3-5 body paragraphs** — each one sentence → short paragraph
- **Payoff** — the one thing you want them to remember
- **Close** — question, CTA, or next-post tease

Aim for 150-400 words. One idea, completed.

## Tutorial / Listicle (Direct Instructional)

**When:** The content IS the list, and the viewer wants a checklist.

**Structure:**
- Hook = the promise ("3 things I'd do if I started over")
- Each numbered item = a self-contained beat with reason + action
- Close = one-line synthesis + CTA

**Rules:**
- Numbers in frame / on screen
- Each item should stand alone (viewer could skip to #3 and still benefit)
- Keep to 3-7 items — more dilutes

## Contrarian / Framework-Flip

**When:** You have a sharp take that challenges a common belief.

**Structure:**
1. State common belief charitably: "Most people think X."
2. Show why it's widely believed: "It's easy to see why."
3. Reveal the flaw: "But here's what that misses."
4. Offer your alternative: "Here's what I do instead."
5. Proof: "I've been doing this for [time] and [result]."
6. CTA or question.

Works especially well on LinkedIn, X, and long-form YouTube.

## Teaching sequence (if teaching a concept)

**When:** Educational content where comprehension matters more than entertainment.

**Structure:**
1. **Why this matters** — stakes for the viewer
2. **What you used to do** / common approach — relatable starting point
3. **Where it breaks down** — the failure mode
4. **The mental model** — the new framing
5. **Concrete example** — apply it
6. **How to start** — first action

Particularly strong for `write-article` and longer YouTube.

## Framework selection heuristic

| Source / intent | Framework |
|---|---|
| Quick take, strong opinion | Contrarian |
| Teaching a concept | Teaching sequence |
| Short-form conversion | PAS |
| Personal story | Hero's Journey (compressed) or StoryBrand |
| Long YouTube | MrBeast retention |
| Newsletter / article | Atomic essay OR Teaching sequence |
| Tutorial / how-to | Tutorial listicle |
| Sponsored segment | AIDA |

## How to use this file

`write-script` reads this to pick a framework. The `brief.md` often specifies which. Skills like `create-thread` and `write-article` adapt these frameworks to their format.
