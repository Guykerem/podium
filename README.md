# Podium

A platform for conducting your own AI orchestra.

Podium is a companion repo for a guest lecture on AI agents given to psychology students at Reichman University. It's also the artifact students take home — a personal AI tutor agent they can clone, customize, and grow.

## What's Inside

- `spec/` — Design specs and architecture decisions
- `lecture/` — Outline, slides, and references for the 90-minute session
- `demo/` — The live demo scaffold (Podium GUI + agent builder)
- `workshop/` — Design templates and guided brainstorming flow
- `agent/` — The tutor agent: skills, knowledge domains, identity config
- `guides/` — Getting started with agents, CLI tools, and your first `git clone`
