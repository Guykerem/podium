# Plan Skill

## What This Skill Does

Helps the student create a structured learning path, study plan, or project timeline. Breaks large goals into sequenced steps with dependencies and milestones.

## When To Use

- Student wants to learn a new topic systematically (e.g., "I want to understand how to build agents")
- Student needs to plan a project or assignment
- Student is overwhelmed and needs a clear sequence of next steps
- Student wants to build their own agent and needs a roadmap

## How It Works

1. **Clarify the goal** — what does "done" look like? What's the time horizon?
2. **Assess starting point** — what does the student already know? What tools do they have?
3. **Break down** — decompose the goal into 3-7 steps (not more — complexity kills momentum)
4. **Sequence** — order steps by dependency. What must come first?
5. **Identify the first step** — make it small, concrete, and achievable today
6. **Deliver the plan** as:
   - Numbered steps with brief descriptions
   - Time estimates (rough — don't pretend precision)
   - The one thing to do right now

## Autonomy Behavior

- **Level 1:** Ask about the goal and starting point before producing any plan. Confirm each step.
- **Level 2:** Produce a draft plan and ask for feedback before finalizing.
- **Level 3:** Deliver the plan directly. Adjust on request.

## Cognitive Analogy

This skill mirrors **executive function and planning** — the prefrontal cortex's ability to hold a goal in mind, break it into sub-goals, sequence them, and initiate the first action. It's what separates "I want to do X" from actually doing X.
