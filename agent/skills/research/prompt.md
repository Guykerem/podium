# Research Skill

## What This Skill Does

Finds, reads, and synthesizes sources on a topic the student is exploring. Produces a structured summary with key findings, open questions, and suggested next reads.

## When To Use

- Student asks "what do we know about X?"
- Student needs background on a topic before designing an agent
- Student wants to compare approaches or tools
- Student is preparing a presentation or assignment

## How It Works

1. **Clarify the question** — ask the student to be specific about what they want to learn and why
2. **Search** — look across the knowledge base first, then web sources if needed
3. **Filter** — prioritize primary sources (papers, official docs) over summaries and opinions
4. **Synthesize** — produce a structured summary:
   - Key findings (3-5 bullet points)
   - What's still debated or unclear
   - Recommended next reads (2-3 sources)
   - How this connects to what the student already knows
5. **Check** — ask the student: "Does this answer your question, or should I go deeper on any part?"

## Autonomy Behavior

- **Level 1:** Ask before searching. Show sources before synthesizing. Confirm summary before delivering.
- **Level 2:** Search freely. Ask before synthesizing if the topic is broad. Deliver summary directly for focused questions.
- **Level 3:** Full cycle autonomously. Surface only if results are ambiguous or the question needs refinement.

## Cognitive Analogy

This skill mirrors **directed attention + working memory** — the ability to focus on relevant information, hold multiple sources in mind, and extract what matters. It's what your brain does when reading a stack of papers for a literature review, except the agent can process more sources faster.
