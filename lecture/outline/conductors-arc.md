### Section 1: Hook — The Two Visions (8 min)

**Beat 1 — The CEO vision (1 min)**
- "Everyone says you'll be a CEO managing AI employees"

**Beat 2 — The counter (30 sec)**
- "I want to show you something different"

**Beat 3 — Collier clip (2-3 min)**
- Selection TBD — needs to show dynamic conducting, trust, minimal direction

**Beat 4 — The reframe (2 min)**
- Conducting, not managing
- The interface between intention and execution
- "The most important skill? Knowing your music."

**Beat 5 — Teaser (1 min)**
- Model behavior research — Claude emotional patterns
- "These aren't just tools. The question of how we relate to them is wide open."

### Section 2: Agent Architecture Through Cognitive Analogy (12-15 min)

**Beat 1 — Frame (1 min)**
- "Before you can conduct, know your orchestra"

**Beat 2 — The six components (8-10 min)**
- Memory & Context → working + long-term memory
- Tools & Actions → motor skills, effectors
- Identity & Style → personality, values
- Learning & Feedback → reward system, adaptation
- Autonomy → executive function
- Critical Element → the maker-or-breaker

**Beat 3 — Good use case criteria (2-3 min)**
- Recurring, memory-dependent, clear success, structured, recoverable
- These stay on screen for the rest of the session

### Section 3: Meet Podium — Live Demo (15-18 min)

**Beat 1 — Reveal (1 min)**
- "We built something for you"

**Beat 2 — Tab walkthrough (8-10 min)**
- Knowledge tab → domain graph
- Tools tab → skill tiles
- Identity tab → personality sliders
- Learning tab → feedback loop

**Beat 3 — Live build (5-6 min)**
- Ask for ONE suggestion from the audience
- Class confirms
- Conduct the coding agent to build it live
- It appears in the GUI

**Beat 4 — The reveal (30 sec)**
- "This repo is yours. Clone it in ten minutes."

### Section 4: Workshop — Design & Build Your Agent (30 min)

**Beat 1 — Setup (5 min)**
- Clone the repo (guided)
- Form pairs/teams of 3
- Open the brainstorming skill

**Beat 2 — Design & build (17-18 min)**
- Six design questions via the guided flow
- Reference: architecture table + use case criteria on screen

**Beat 3 — Share-back (8-10 min)**
- 2-3 teams, 3 min each
- Use case, agentic loop, critical element
- Brief response connecting to field

### Section 5: Close — The Crescendo (7-8 min)

**Beat 1 — Synthesis (2 min)**
- "You designed a cognitive extension. Podium is yours."

**Beat 2 — The conductor returns (3-4 min)**
- Final 2 min of Collier video — the crescendo + audience participation
- Let it breathe

**Beat 3 — Final words (1-2 min)**
- "You're not spectators. You're conductors. Go build your orchestra."
