# Lecture

The 90-minute guest session on AI agents — given to psychology students at Reichman University, designed to be reusable for anyone running a similar talk.

## Files

### `outline/`
- [`conductors-arc.md`](./outline/conductors-arc.md) — the spine of the talk. The conductor-not-CEO metaphor, beat by beat, with timing.
- [`deck-outline.md`](./outline/deck-outline.md) — slide-by-slide breakdown ready to pour into Keynote/Slides.
- [`title-options.md`](./outline/title-options.md) — candidate session titles.

### `references/`
- [`course-context.md`](./references/course-context.md) — the surrounding course this lecture lives inside, learning objectives, what students already know.
- [`student-profiles.md`](./references/student-profiles.md) — who's in the room. Use this to calibrate density and examples.

### `speaker-notes.md`
The full speaker script — what to say, when to pause, what to demo live, how to land the metaphor.

### `slides/`
Reserved for the rendered deck. Not in repo (rebuild from `outline/deck-outline.md`).

## How to use this

If you're **attending** the lecture: nothing required. Show up.

If you're **running** the lecture (or a variant):
1. Read `outline/conductors-arc.md` for the shape.
2. Skim `references/student-profiles.md` to calibrate.
3. Read `speaker-notes.md` for the prose-level detail.
4. Generate slides from `outline/deck-outline.md`.

If you're **studying after** the lecture: the `workshop/` directory is your next step — design your own role using the six-question template, then have the `agent-architect` role help you scaffold it.
